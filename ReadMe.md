# STimer

A wxWidgets GUI trinket on top of OS's `shutdown` program. Pronounced ASS timer.

Features:

- Combobox to select shutdown hour and minute (Whoah!)
- Utterly useless

## Prerequisites

- Visual Studio 2019, GCC or Clang
- libgtk-3-dev on Linux
